# 🛍️ Digital Store

Projeto de e-commerce desenvolvido em **React**, utilizando **Vite** e **TailwindCSS** para construção de uma interface moderna e responsiva.

A aplicação simula uma loja virtual com listagem de produtos, filtros por categoria, destaque de ofertas e navegação entre páginas.

---

## 🚀 Tecnologias utilizadas

- ⚛ React
- ⚡ Vite
- 🎨 TailwindCSS
- 🔗 React Router DOM
- 🎯 React Icons

---

## 📸 Preview

Interface principal da loja com:

- Barra de navegação
- Campo de busca
- Filtro de produtos
- Cards de produtos
- Destaques de ofertas
